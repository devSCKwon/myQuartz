사고 발생 현황 및 원인

사고 발생 개요

외부에서 사내 임직원 메일계정이 스팸메일 발송에 악용되어, 회사도메인 차단

  • 외부 IP에서 취약한 사용자 계정으로 로그인을 시도하여, 로그인이 성공한 뒤 사내메일로 피싱메일 발송에 이용되었습니다.
  • APT 방어차단 회피 목적으로 여러차례 분할하여 발송되었으며, 로그인 후 사용자 사칭 및 세금계산서 전송과 같은 내용으로 스팸메일을 발송한 것으로 확인되었습니다.

실제 사고사례

  • 2025년 5월 22일: 품질보증팀 책임매니저 계정 - 외부아이피를 통한 아이디 / 패스워드 웹 접속 : 사용자 사칭(사용자명 변경) 스팸메일 발송
  • 2025년 7월 15일: 외주관리팀 책임매니저 계정 - 외부아이피를 통한 아이디 / 패스워드 웹 접속 : 사용자 사칭(사용자명 변경) 및 당사정보 이용(메일 제목) 스팸메일 발송
  • 2025년 8월 5일: 물류팀 책임매니저 계정 - 외부아이피를 통한 아이디 / 패스워드 웹 접속 : 당사정보 이용(사용자명 이용)한 세금계산서(악성웨어 첨부) 스팸메일 발송

주요 유출 경로 및 위험 요소

가장 핵심적인 보안 위협은 타 사이트에서 이미 유출된 계정 정보의 재사용입니다.

해커들은 다른 웹사이트에서 유출된 아이디와 비밀번호를 활용하여 회사 메일시스템 접속을 시도합니다. 동일한 계정 정보를 여러 사이트에서 사용할 경우, 한 곳에서 정보가 유출되면 연쇄적으로 다른 시스템까지 침해당할 위험이 높습니다.

이 외에도 피싱메일을 통한 계정 정보 탈취, 악성코드 감염, 약한 비밀번호 사용 등이 주요 위험 요소입니다.

계정 보안 강화 필수 조치사항

비밀번호 관리 원칙

  • 타 사이트와 절대 동일한 비밀번호를 사용하지 마십시오.
  • 회사 메일계정 전용 비밀번호를 별도로 설정해야 합니다.
  • 현재 메일시스템은 8자리 이상의 영문 대소문자, 숫자 조합을 최소 요구사항으로 적용하고 있습니다.
  • 보안 강화를 위해 특수문자를 포함한 복잡한 비밀번호 사용을 권장합니다.
  • 비밀번호 변경주기가 30일로 강화 적용되었습니다.

비밀번호 정책 추가 (2025년 8월 5일 이후 추가된 강화정책 적용)

다중인증(MFA) 설정

필요시 메일계정에 2단계 인증을 활성화하시기 바랍니다. 설정 방법은 다음과 같습니다:

  • 환경설정 > 보안 > 로그인설정 > 2단계 인증(선택) : 활성화
  • 인증방법: 이메일 또는 휴대폰 문자(SMS)로 일회용 인증번호 수신
  • 개인이메일이 등록되지 사용자가 많습니다. 반드시 본인이 확인하시고 설정하시기 바랍니다.
    • MFA 설정
    MFA 로그인방법
    MFA 2차인증 입력

개인 이메일 주소와 휴대폰 번호로 일회용 인증번호를 받아 입력하는 방식입니다. 의심스러운 활동이 감지된 대상자는 강제 2단계 인증을 적용할 예정입니다.

  • 2025년 8월 기준 비정상 로그인확인 대상자 12명 강제 적용
  • 2025년 9월 이후, MFA 2차인증을 강제화 하는 방안을 검토, 향후 시행시 공지 후에 적용

의심스러운 활동 즉시 신고

  • 로그인 알림 메일을 받은 경우 접속 이력을 반드시 확인하시기 바랍니다.
  • 본인이 접속하지 않은 로그인 기록 발견 시 즉시 전산팀으로 신고하시기 바랍니다.
  • 접속현황 확인방법: 환경설정 > 보안 > 접속현황
  • ※ 참고 : 접속현황 IP (211.228.165.250) 은 세진 IP 입니다.
    • 접속현황 확인방법
    비정상접속 의심사례
  • 의심스러운 메일 수신 시 링크 클릭이나 첨부파일 실행을 금지합니다.

즉시 실행 사항 및 예상 피해

즉시 실행 사항

  • 현재 사용 중인 메일계정 비밀번호가 다른 사이트와 동일한지 점검하고, 동일한 경우 즉시 변경하시기 바랍니다.
  • 2단계 인증이 설정되지 않은 계정은 필요시 설정을 완료하시기 바랍니다.
  • 최근 3개월 내 메일 로그인 이력을 확인하여 의심스러운 접속 여부를 점검하시기 바랍니다.

사고 발생 시 예상 피해

메일계정이 침해당할 경우 사내 신뢰도를 악용한 피싱 공격, 메일 정보 유출, 회사 평판 손상 등의 2차 피해가 발생할 수 있습니다. 특히 사내 메일계정을 통해 발송된 악성메일은 수신자의 신뢰를 바탕으로 피해를 확산시킬 가능성이 높습니다.

타 사이트 유출 정보 재사용으로 인한 계정 침해는 예방 가능한 보안 사고입니다. 모든 임직원께서는 위 조치사항을 철저히 준수하여 회사 정보보안 강화에 적극 협조해 주시기 바랍니다.
