사내 문서 보안 강화 활동

시행일: 2025년 7월 21일(월)

그룹웨어 기안 서식 변경

배경 및 목적

  • 목적: 전사적인 문서 보안 수준을 한 단계 높이고, 문서 관리 체계를 표준화하여 회사의 정보 자산을 더욱 체계적이고 안전하게 보호하고자 함.
  • 주요 변경 사항: 기존에 가장 보편적으로 사용되던 '일반 기안지'를 비활성화하고, 문서의 중요도에 따라 보안 등급 설정이 의무화된 신규 서식 2종을 도입함.

주요 변경 내용: AS-IS → TO-BE

구분 AS-IS (기존) TO-BE (변경 후)
기안 서식 '일반 기안지' 서식을 통해 모든 일반/대외비 문서 작성 기존 '일반 기안지' 서식 사용 비활성화
보안 설정 별도의 보안 등급 설정 절차 없음 기안 시 보안 등급(일반, 대외비, 비밀) 설정 의무화
기안 방식 단일 서식 사용 모든 기안 업무는 신규 '문서등급 기안지' 2종을 목적에 맞게 사용

신규 보안 서식 사용 안내

2025년 7월 21일(월)부터 모든 기안은 아래의 신규 서식 2종 중 목적에 맞는 서식을 선택하여 작성해야 함.

구분 문서등급 (대외비) 기안지 문서등급 (비밀) 기안지
주요 용도 기존 '일반 기안지'의 모든 업무 대체 (일반, 대외비 문서 작성) '비밀' 등급 문서 작성 전용
작성 방법 '보안 등급' 필드에서 '대외비' 또는 '일반' 선택 '보안 등급' 필드에서 '비밀' 등급만 선택 가능
보안 특징 기존과 동일 결재자를 제외한 수신/공유/재공유 기능 원천 차단
내용 공유 수신, 참조, 공유 기능 사용 가능 내용 공유 필요 시, '참조' 기능을 통해 결재 라인에 직접 추가하는 방식만 가능

임직원 요청 사항

  • 시행일: 2025년 7월 21일(월)부터 모든 기안 업무 시, 반드시 상기 신규 서식 2종 사용.
  • 작성자: 문서의 민감도를 신중하게 판단하여 적절한 비밀등급 지정 필요.
  • 결재자: 문서의 민감도를 최종 판단하고, 등급이 잘못 지정된 경우 반려하여 올바른 등급으로 재작성하도록 안내.

사내 메일 보안 강화

메일 보안 등급 종류

  • 사내 메일 발송 시, 관리자가 설정한 보안 등급(일반, 대외비, 기밀)을 선택하여 중요한 메일을 안전하게 전송 가능.
  • 보안등급이 메일 상단에 표기되며, 등급에 따라 지정된 보안 문구가 바닥글에 자동 반영됨.
  • (예: - 기밀 / 본 문서는 ㈜세진의 내부문서이며, 외부유출 및 재전송을 금합니다.)

'대외비' 및 '기밀' 등급 메일 특징

  • 외부 발송 불가: '대외비' 또는 '기밀' 등급으로 설정된 메일은 외부 주소로 발송이 불가능함.
  • 보안 설정 유지: 해당 등급의 메일을 회신/전달 시, 기존 보안 등급, 유효 기간, 재전송 금지 옵션을 임의로 수정하거나 삭제할 수 없음.
  • 유효 기간 설정: 메일 유효 기간(하루/일주일/1개월) 설정 가능하며. 유효 기간 만료 시 메일은 받은 사람의 메일함에서 자동으로 삭제되어 열람 불가능.
  • 재전송 금지: '대외비' 등급은 재전송 금지를 옵션 선택 가능, '기밀' 등급은 옵션 선택과 관계없이 재전송 기능이 원천 차단됨.

문서 공유 및 보안 관리 가이드

문서 공유 기본 원칙

  • 문서의 민감도와 활용 목적에 따라 열람 권한과 공유 방식을 설정하는 것이 중요함.
  • 작성자는 문서 내용을 조회할 수 있는 대상을 명확히 인지하고 적절한 공유 방법을 선택해야 함.

문서 열람/공유 방법 및 특징

방법 설명 예시 특이사항
결재라인 (참조) 기안 or 결재권자에 의해 참조자로 지정된 사용자는 결재선에 포함되어 열람 가능 그룹웨어 극비/대외비 샘플 서식, 영업비밀 등급 자체확인 서식 결재라인에 존재하나 별도 승인 절차 없음
열람문서 기능 서식 내 열람 권한 내장. 결재라인 외에도 열람 가능 지출품의서 (재경부 확인 목적) 결재문서 리포트 통계 활용 시 열람 권한 필수
문서공유 특정 부서 또는 사용자에게 문서 직접 공유 작성부서의 정책안내문을 타부서에 공유 실시간 협업에 적합, 공유 시점 및 범위 통제 가능
수신함 지정 결재선과 무관하게 업무 수행자(팀 또는 개인) 지정 가능 협조전 수신 → 재기안 필요, 팀 단위 수신 시 담당자 지정 수신자별 완료 이력 기록 가능

지속적인 보안 강화 계획

  • 기능 확대: 공용 서식 내 '비밀' 등급 문서의 보안 강화를 위한 기능 점진적 고도화.
  • 권한 체계 수립: 서식 유형 및 직무(Role-Based) 기반 명확한 권한 체계 수립으로 불필요한 접근 원천 차단. 또한, 열람/공유/수정 등 모든 활동 이력에 대한 책임 소재 명확화.
  • 가이드 및 교육 제공: '결재선 이해하기' 등 관련 가이드 지속 제공으로 임직원 보안 의식 제고.
